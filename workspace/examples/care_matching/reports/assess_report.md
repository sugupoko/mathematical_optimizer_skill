# アセスメントレポート — 介護マッチング最適化

## 1. 問題概要

介護サービスにおける利用者（15名）とヘルパー（10名）の割当最適化問題。
東京都5区（杉並区・世田谷区・渋谷区・新宿区・中野区）を対象とする。

## 2. データ概要

| 項目 | 値 |
|------|-----|
| 利用者数 | 15名 |
| ヘルパー数 | 10名 |
| 総受入可能数 | 40名分 |
| 供給/需要比 | 2.67倍 |

供給が需要の約2.7倍あり、**全員マッチング可能**な環境。ただし資格・経験・地区の制約で実質的な選択肢は限られる。

## 3. 地区別分布

| 地区 | 利用者数 | ヘルパー数 |
|------|---------|-----------|
| 杉並区 | 4 | 2 |
| 世田谷区 | 3 | 2 |
| 渋谷区 | 2 | 2 |
| 新宿区 | 3 | 2 |
| 中野区 | 3 | 2 |

各区にヘルパーが2名ずつ配置されており、比較的均等。杉並区のみ利用者がやや多い。
隣接区間のマッチングも許容されるため（杉並-世田谷-渋谷-新宿-中野の隣接関係）、地区制約は柔軟に対応可能。

## 4. 要介護度分布

| 要介護度 | 人数 | 特記 |
|---------|------|------|
| 1 | 2名 | 制約が軽い |
| 2 | 3名 | 制約が軽い |
| 3 | 4名 | 中程度 |
| 4 | 3名 | 経験3年以上が必須（HC02） |
| 5 | 3名 | 経験3年以上必須 + 10年以上推奨 |

要介護4以上が6名（40%）おり、経験豊富なヘルパーへの需要が高い。

## 5. 資格・サービス分析

- **介護福祉士が必要な利用者**: 7名（身体介護を含むサービス）
- **介護福祉士を持つヘルパー**: 7名
- **初任者研修のヘルパー**: 3名（C004, C006, C009）

資格面では介護福祉士7名 vs 必要7名でぴったり。初任者研修の3名は生活援助のみの利用者に限定される。

## 6. 性別希望

- 女性希望: 5名（R001, R003, R006, R010, R012）
- 希望なし: 10名
- 女性ヘルパー: 6名、男性ヘルパー: 4名

女性希望5名に対して女性ヘルパー6名と十分だが、資格・地区の制約と合わせると選択肢が限定される可能性あり。

## 7. 継続性（満足度5の組合せ）

信頼関係が構築済みの7ペア:

| 利用者 | ヘルパー | 備考 |
|--------|---------|------|
| R001(田中花子) | C001(森田美咲) | 長期担当、入浴介助が良い |
| R002(鈴木一郎) | C002(岡田健一) | 同地区、状態把握 |
| R003(佐藤美智子) | C010(長谷川あかり) | 渋谷区内、丁寧 |
| R004(高橋正男) | C008(橋本由紀) | 要介護5対応経験豊富 |
| R007(渡辺勝) | C002(岡田健一) | 世田谷区内、技術高い |
| R009(小林清) | C008(橋本由紀) | 重度対応で最も信頼 |
| R010(加藤節子) | C005(前田恵美) | 中野区内、継続担当 |

**注意**: C002がR002・R007の両方と満足度5、C008がR004・R009の両方と満足度5。全7ペアの同時維持には競合がある。

## 8. ハードコンストレイント

| ID | 内容 | 影響 |
|----|------|------|
| HC01 | 身体介護には介護福祉士必須 | 150ペア中28ペアが不可 |
| HC02 | 要介護4以上は経験3年以上 | C004(2年), C009(1年)が該当不可 |
| HC03 | 担当数上限 | 各ヘルパーに上限あり |
| HC04 | 曜日重複1日以上 | スケジュールが合わないペアを除外 |
| HC05 | R004-C004はNG | 過去トラブル |

実行可能ペア: 150ペア中 **122ペア（81.3%）**

## 9. 仮説と主要リスク

1. **全員マッチングは達成可能** — 供給が十分にある
2. **ボトルネックはC002とC008の競合** — 満足度5ペアが集中しており、全維持は容量制約次第
3. **初任者研修ヘルパー（C004, C006, C009）の活用がカギ** — 生活援助のみの利用者（R005, R008, R011, R014）に効率的に配置する必要あり
4. **公平性は意識的に最適化しないと偏る** — 貪欲法だとC009が未配置になりやすい
