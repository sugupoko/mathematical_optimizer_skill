# 改善レポート

## 改善策: AM/PM分割（2便運行）

### 背景
- 総需要815 kg > 総容量750 kg のため、1便では全顧客をカバーできない
- OR-Tools最適解でも1顧客が未訪問

### AM/PM分割の設計

時間枠の中間値で顧客をAM/PMに分類:
- **AM便**（08:00出発）: 時間枠中間値が12:00以前の16顧客（635 kg）
- **PM便**（12:30出発）: 時間枠中間値が12:00以降の4顧客（180 kg）

各便を独立にOR-Tools Routing Solverで最適化。

### 結果

| 指標 | OR-Tools 1便 | AM/PM分割 | 変化 |
|------|-------------|----------|------|
| 総距離 | 84.5 km | 105.8 km | +25.2%（2便分の移動） |
| 総コスト | 9,508円 | 11,709円 | +23.1% |
| 時間枠違反 | 0 | 0 | - |
| 容量違反 | 0 | 0 | - |
| **未訪問** | **1** | **0** | **完全配送達成** |

### 分析

1. **全顧客への配送を達成**: 最も重要な改善。未訪問0件を実現
2. **コスト増は+23%**: 2便運行により移動距離は増加するが、配送完了率100%の価値は大きい
3. **時間枠違反なし**: AM/PM分割により各便の時間的余裕が増加
4. **容量違反なし**: AM 635kg / PM 180kg と、各便で容量内に収まる

### トレードオフ

- 距離・コストは1便より増加（デポ往復が増えるため）
- ドライバーの拘束時間が延びる可能性（特にAM+PM両方担当する場合）
- PM便の顧客が少なく（4件）、車両の稼働率が低い

### 追加改善の方向性

1. **地理的クラスタリング**: AM/PM分割に加え、エリア別に車両を割当
2. **需要量ベースの分割**: 時間枠だけでなく需要量のバランスも考慮
3. **車両追加の検討**: 小型車両1台の追加でコスト効率が改善する可能性
