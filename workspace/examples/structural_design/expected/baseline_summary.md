# ベースライン比較結果 — 片持ち梁の構造最適化

## 問題規模
- 構造タイプ: 片持ち梁（固定端-自由端）
- 全長: 2.0 m
- 材料: SS400（E=210GPa, 密度7850kg/m^3, 降伏応力245MPa）
- 荷重: 先端30kN + 中央5kN = 合計35kN
- 設計変数: 幅(b)と高さ(h)の2変数

---

## 手法別結果

### 1. 均一断面（ベースライン）

最大寸法で設計した場合（安全だが重い）。

| 指標 | 値 |
|------|-----|
| 幅 x 高さ | 0.30 x 0.50 m |
| 重量 | **2,355 kg** |
| 最大応力 | 7.0 MPa |
| 最大たわみ | 0.011 mm |
| 応力比 | 0.04 (許容の4%) |
| たわみ比 | 0.002 (制限の0.2%) |
| 判定 | 過剰設計 |

明らかに過剰。応力利用率がわずか4%であり、材料の96%が無駄。

### 2. 最小断面（フィージビリティ確認）

最小寸法で設計した場合（軽いが制約違反の可能性）。

| 指標 | 値 |
|------|-----|
| 幅 x 高さ | 0.05 x 0.10 m |
| 重量 | **78.5 kg** |
| 最大応力 | 2,100 MPa |
| 最大たわみ | 160 mm |
| 応力比 | 12.86 (**違反**) |
| たわみ比 | 32.0 (**違反**) |
| 判定 | 不可能 |

応力が許容値の12.86倍、たわみが制限の32倍。全く成立しない。

### 3. SLSQP最適化（目標: 重量最小化）

| 指標 | 期待値 |
|------|--------|
| 幅 x 高さ | 最適化により決定 |
| 重量 | 100-300 kg 程度（ベースラインの5-15%） |
| 応力比 | 1.0に近い（制約アクティブ） |
| たわみ比 | 1.0に近い（制約アクティブ） |
| 判定 | 実行可能 & 軽量 |

最適解では応力制約またはたわみ制約（あるいは両方）がアクティブになるはず。
つまり「これ以上軽くすると制約に違反する」ギリギリの設計になる。

## ボトルネック分析

### 主要ボトルネック: たわみ制約

**予測:**
片持ち梁の場合、たわみは L^3 に比例するため、たわみ制約が応力制約より先にアクティブになることが多い。
つまり「強度は余裕があるが、たわみが大きすぎる」という状況。

**改善の方向性:**
1. **高さを優先的に大きくする**: たわみは h^3 に反比例（断面二次モーメント I = bh^3/12）。高さを2倍にすればたわみは1/8になる。
2. **トポロジー最適化で材料配置を最適化**: 均一断面ではなく、応力が集中する固定端付近に材料を集中させる。
3. **たわみ制約の緩和検討**: L/400 は厳しめ。搬送ラインの要求を再確認し、L/250 で十分ならば大幅に軽量化できる可能性がある。

## トポロジー最適化の期待結果

80x30要素、体積率40%で実行した場合:
- 片持ち梁の典型的なトポロジー: 上下フランジ + 対角ウェブ（三角トラス状）
- 固定端付近に材料が集中し、自由端付近は薄い
- コンプライアンスはフル材料の1.5-2.5倍程度に増加（体積率40%なので当然）
