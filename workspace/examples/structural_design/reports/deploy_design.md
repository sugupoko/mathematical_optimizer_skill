# 構造最適化 -- 運用設計書

**作成日**: 2026-03-30
**対象**: 工場搬送ライン用片持ち梁（全長2.0m、SS400）の断面・トポロジー最適化

---

## 1. 実行概要

| 項目 | 設定 |
|------|------|
| 実行頻度 | **プロジェクト単位**（新規構造設計時） |
| 計算時間 | 断面最適化 ~1秒 + トポロジー最適化 ~20秒 |
| データソース | JSON（`data/structure.json`, `data/constraints.json`） |
| 出力 | 最適設計JSON + トポロジー密度配列（.npy） |
| 実行スクリプト | `scripts/run_pipeline.py` |

### 利用シーン

本パイプラインは定期実行ではなく、設計ツールとして使用する。

```
新規構造の設計依頼
  -> structure.json / constraints.json を作成
  -> run_pipeline.py を実行
  -> 結果を確認し、標準サイズへの丸め等を適用
```

---

## 2. パイプライン構成

```
Step 1: データ読み込み（structure.json + constraints.json）
  |
Step 2: バリデーション（材料特性・荷重・制約の妥当性チェック）
  |
Step 3: 断面最適化（scipy SLSQP で重量最小化）
  |
Step 4: トポロジー最適化（SIMP法で材料配置を最適化）
  |
Step 5: 結果検証（応力 <= 許容応力、たわみ <= 制限値）
  |
Step 6: 最適設計JSON + トポロジー密度配列出力
```

## 3. バリデーションルール

| チェック項目 | 条件 | アクション |
|------------|------|----------|
| ヤング率 E | > 0 | **中断** |
| 密度 | > 0 | **中断** |
| 降伏応力 | > 0 | **中断** |
| 梁の全長 | > 0 | **中断** |
| 荷重の大きさ | > 0（各荷重） | **中断** |
| 安全率 | >= 1.0 | **中断** |
| 最大たわみ | > 0 | **中断** |
| 寸法範囲 | min < max（幅・高さ） | **中断** |
| 体積率 | 0 < vf < 1 | **中断** |
| メッシュサイズ | nx, ny >= 2 | **中断** |

---

## 4. 最適化手法

### 断面最適化（SLSQP）

- **設計変数**: 矩形断面の幅(b)と高さ(h)
- **目的関数**: 重量（= b x h x L x 密度）の最小化
- **制約**: 応力 <= 許容応力、たわみ <= 制限値、寸法範囲
- **計算時間**: 約0.01秒（10回程度の反復）

### トポロジー最適化（SIMP法）

- **設計変数**: 各要素の密度（0-1の連続値）
- **目的関数**: コンプライアンス（変形エネルギー）の最小化
- **制約**: 体積率制約
- **計算時間**: 約5-10秒（30-50反復）

### 期待される結果（現行データ基準）

| 指標 | 値 |
|------|-----|
| 最適幅 | 50.0 mm |
| 最適高さ | 113.4 mm |
| 重量 | 89.0 kg（最大断面比 96%削減） |
| 応力比 | 100%（制約がアクティブ） |
| たわみ比 | 91.5% |

## 5. 監視指標

設計結果の品質を以下で確認する。

| 指標 | 正常範囲 | 注意条件 |
|------|---------|---------|
| 応力比 | 0.5-1.0 | > 1.0 は制約違反（設計不可） |
| たわみ比 | 0.5-1.0 | > 1.0 は制約違反（設計不可） |
| 重量削減率 | 50%以上 | 30%未満は制約が厳しすぎる可能性 |
| SLSQP 収束 | success=True | False は初期値やスケーリングを見直す |
| トポロジー収束 | 50反復以内 | 100反復到達は r_min やペナルティを調整 |

---

## 6. 注意事項

### 製造への反映手順

最適化結果をそのまま製造に使用しないこと。

1. 断面最適化の結果を標準サイズに丸める（constraints.json の standard_sizes 参照）
2. トポロジー結果はトラス化・肉抜き加工の参考情報として使用
3. 丸め後の設計を再評価し、制約を満たすことを確認

### 前提条件

| 項目 | 値 | 備考 |
|------|-----|------|
| 荷重条件 | 静的のみ | 動的荷重は安全率1.5で考慮 |
| 材料 | SS400 | 単一材料。複合材は対象外 |
| 座屈 | 未考慮 | 薄い断面では別途評価が必要 |
| 接合部 | 応力集中無視 | ボルト穴等は別途評価が必要 |

## 7. ファイル構成

```

  data/
    structure.json       <- 構造定義（梁の寸法・材料・荷重）

  scripts/

    solve_all.py         <- 分析用（初回のみ使用）
  results/
    design_*.json        <- 最適設計データ
    topology_*.npy       <- トポロジー密度配列
    all_results.json     <- 全手法比較結果
  reports/
    v1_proposal.md       <- 改善提案書
    deploy_design.md     <- この運用設計書
  log/
    run_*.log            <- 実行ログ
  .opt_state.yaml        <- スキル間状態管理
```
