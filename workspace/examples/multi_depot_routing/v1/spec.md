## 目的

- 主目的: 全 13 HC を満たす feasible な週次配送計画
- 副目的: SC1-SC6 の総合スコア最大化 (距離最小化・公平性・優先配送等)

## ハード制約

| # | 制約 | 根拠 | Ref | 備考 |
|---|------|------|-----|------|
| HC1 | 各顧客は計画週内に 1 回だけ訪問 | 業務要件 | R4, R5 | 30 顧客 |
| HC2 | 車両の重量容量を超えない (日次) | 物理 | R2, R5 | 500-2000 kg |
| HC3 | 車両の体積容量を超えない (日次) | 物理 | R2, R5 | 3-10 m³ |
| HC4 | 車両の日次最大稼働時間を超えない | 労務 | R2, R5 | 480-540 分 |
| HC5 | 顧客の必須車両タイプに一致する車両のみ | 業務 | R2, R4, R5 | refrigerated 等 |
| HC6 | ドライバーは顧客の必須認定資格を全て持つ | 法規 | R3, R4, R5 | dangerous_goods 等 |
| HC7 | ドライバー認定が車両タイプに対応 | 法規 | R2, R3, R5 | 冷蔵車 → refrigerated 認定 |
| HC8 | ドライバーの利用不可日に割当しない | 従業員都合 | R3, R5 | |
| HC9 | 車両の利用不可日に割当しない | 整備日程 | R2, R5 | |
| HC10 | 車両はホームデポから出発・帰着 | 運用 | R1, R2, R5 | |
| HC11 | ドライバーの週最大勤務時間を超えない | 労務 | R3, R5 | 45-50 時間 |
| HC12 | 同日に 1 人のドライバーは 1 台の車両のみ | 運用 | R5 | |
| HC13 | 顧客の時間枠要望 (morning/afternoon) を守る | 顧客要求 | R4, R5 | |
