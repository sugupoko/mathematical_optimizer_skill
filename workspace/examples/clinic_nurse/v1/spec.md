## 受け入れ要件

### 現場の管理方法
- 現在のツール: Excel（3クリニックそれぞれで手動作成）
- 操作する人: 各クリニックの事務長（3名）
- 良い点: クリニックごとに独立して管理できる
- 不満点: 看護師が複数クリニックを掛け持ちする場合の調整が手作業で大変、公平性が見えない
- 属人化: あり（事務長の経験頼み）

### 結果の表示
- 希望する形式: クリニック別の週次シフト表（現行 Excel 形式を踏襲）
- 追加で欲しい情報: 看護師ごとの勤務時間サマリ、夜勤回数、home_clinic 適合率
- 色分け: クリニック別（C1=青、C2=緑、C3=紫）
- 閲覧環境: PC（Excel）→ 印刷して各クリニック掲示

### 手修正
- 頻度: 週1-2回（欠勤・希望変更）
- 制約チェック: 欲しい（特に HC4 資格マッチ、HC7 senior 配置）
- ドラッグ&ドロップ: あれば嬉しい（v2 以降）

### 承認・配布フロー
- 各クリニック事務長が作成 → 看護部長が全体確認 → 印刷配布
- 確定後の変更: 週1-2回（当日朝の欠勤対応）

### 更新頻度
- 作成: 隔週金曜に次2週間分を作成
- 急な変更: 当日朝の欠勤連絡に対応（代替要員を手動で探す）

### 納品形態
- Phase 1: Excel書き戻し（クリニック別シート + 全体サマリ）
- Phase 2: 手修正時の制約チェック機能
- Phase 3: Web 化は現時点では不要
