# ベースライン比較結果 — 施設配置最適化（関東エリア倉庫配置）

## 問題規模
- 候補施設: 10箇所（埼玉2、千葉2、神奈川2、東京西部2、茨城1、群馬1）
- 顧客: 30小売店（関東広域に分布）
- 総需要: 3,820 units/月
- 総容量: 10,200 units/月（全候補の合計）
- カバレッジ要件: 全顧客が最寄り倉庫から50km以内

---

## 手法別結果

### 1. 全施設開設 + 最寄り割当
| 指標 | 値 |
|------|-----|
| 総コスト | **8,188,436 円/月** |
| 固定費 | 8,100,000 円/月 |
| 輸送費 | 88,436 円/月 |
| 開設施設数 | 10 |
| 平均距離 | 19.6 km |
| 最大距離 | 51.5 km |
| 容量違反 | 1件（W01: 1860 > 1500） |

固定費が総コストの99%を占める。輸送距離は最短だが、コスト効率が極めて悪い。
W01（さいたま物流センター）に需要が集中し容量違反が発生。

### 2. 貪欲法（コスト削減量ベースの逐次追加）
| 指標 | 値 |
|------|-----|
| 総コスト | **849,871 円/月** |
| 固定費 | 600,000 円/月 |
| 輸送費 | 249,871 円/月 |
| 開設施設数 | 1（W07: 八王子倉庫） |
| 平均距離 | 55.5 km |
| 最大距離 | 168.7 km |
| 容量違反 | 1件（W07: 3820 > 800） |

固定費が支配的なため、1施設で打ち止めになった。
容量を4.8倍超過しており、実運用不可能。

### 3. PuLP UFL ソルバー（容量制約なし）
| 指標 | 値 |
|------|-----|
| 総コスト | **849,871 円/月**（最適解） |
| 固定費 | 600,000 円/月 |
| 輸送費 | 249,871 円/月 |
| 開設施設数 | 1（W07: 八王子倉庫） |
| 平均距離 | 55.5 km |
| 最大距離 | 168.7 km |
| 容量違反 | 1件（容量を考慮しない定式化） |
| 求解時間 | 0.06秒 |

貪欲法と同一解に到達。UFLでは容量制約がないため、固定費が安い1施設に全顧客を集約するのが最適。
**UFL単独では実用的な解が得られないことが確認された。**

## ボトルネック分析

### 主要ボトルネック: 固定費と輸送費のトレードオフ

**原因:**
1. 固定費（50万〜120万円/月）が輸送費（距離×150円/km）に比べて大きい
2. 施設を1つ追加するコスト（50万円以上）が、輸送費の削減額を上回る
3. その結果、UFL・貪欲法とも「1施設に全集約」が最適になってしまう

**影響:**
- 容量を無視すると非現実的な解になる（1施設に3820 units、容量800）
- 全施設開設すると固定費だけで810万円/月
- 現実的な解は「容量制約あり（CFL）」で探す必要がある

### カバレッジの課題
- 水戸ショップ（S21）は全候補施設から50km超（最寄り: 土浦倉庫 51.5km）
- 50kmカバレッジ率は最良でも97%（29/30）
- 水戸エリアに追加候補地が必要か、土浦倉庫の活用を検討すべき

### 改善の方向性
1. **CFL（容量制約付き）**: 容量を守りつつコストを最小化
2. **P-median**: 施設数を固定して距離の公平性を重視
3. **カバレッジ制約の追加**: 50km上限を明示的にモデルに組み込む
4. **感度分析**: 固定費の変動に対する解の安定性を検証
