# 施設配置最適化 提案書 v1

## エグゼクティブサマリー

関東エリアにおける倉庫配置を最適化しました。
10候補地から**4倉庫を開設**し、30小売店への配送を**容量違反なし**で実現します。
月額コストは約312万円で、全施設開設（819万円）に対して**62%のコスト削減**です。

## 1. 現状の課題

### 意思決定の複雑さ

| 項目 | 数値 |
|------|------|
| 候補施設 | 10箇所 |
| 小売店 | 30店舗 |
| 組合せ数 | 2^10 × 30^10 ≒ 事実上無限 |
| 月間需要 | **3,820 units** |
| 必要容量 | 最低3施設分 |

「どこに倉庫を置くか」の判断は、固定費・容量・距離の3要素が絡み合い、直感では最適解が見えません。

### カバレッジの制約

- 全30店舗を50km以内でカバーしたい
- しかし水戸ショップ（S21）は全候補から50km超
- 現在の候補地では100%カバレッジは達成不可能

## 4. 注意事項とリスク

### 4.1 稼働率の偏り

4施設とも稼働率99-100%で運用される。これは以下のリスクを伴う:

- **需要増加への対応余地がない**: 需要が5%増えると容量超過
- **障害時の代替がない**: 1施設停止で全顧客の25%に影響

**緩和策**: 5施設体制にする（月額+45万円で余裕が生まれる）

### 4.2 カバレッジの限界

最大距離104.8kmの顧客が存在する。50km要件を満たすには:

1. **候補地追加**: 水戸・つくば近辺に候補を追加して再最適化
2. **要件緩和**: 50km → 55kmに緩和すればほぼ全店カバー
3. **別ルート対応**: 遠距離店舗は直送や共同配送で対応

### 4.3 データの精度

| 仮定 | 現状 | 改善方法 |
|------|------|---------|
| 距離 | Haversine×1.3の概算 | Google Maps API で実走行距離に置換 |
| 輸送コスト | 150円/km固定 | 車種・頻度別の実績コストで置換 |
| 需要 | 月間固定値 | 季節変動を考慮した年間計画に拡張 |

## 付録: 用語説明

| 用語 | 説明 |
|------|------|
| UFL | Uncapacitated Facility Location。容量制約なしの施設配置問題 |
| CFL | Capacitated Facility Location。容量制約付きの施設配置問題 |
| P-median | ちょうどP箇所の施設を開設して距離合計を最小化する問題 |
| 稼働率 | 割り当てられた需要 / 施設の容量。100%超は容量違反 |
| カバレッジ | 指定距離以内に施設がある顧客の割合 |
