# 改善レポート: 施設配置最適化

## 1. 改善概要

ベースライン（UFL: 1施設全集約、容量違反あり）から、以下の3改善策を実施した。

1. **CFL ソルバー**: 容量制約を追加
2. **P-median（P=3,4,5）**: 施設数を固定して距離の公平性を検証
3. **CFL + カバレッジ制約**: 50km上限を追加

## 2. 改善手法と結果

### 改善1: CFL ソルバー（容量制約付き）

| 指標 | UFL（ベースライン） | CFL（改善） | 差分 |
|------|-------------------|------------|------|
| 総コスト | 849,871 円/月 | **3,124,407 円/月** | +2,274,536 |
| 施設数 | 1 | **4** | +3 |
| 容量違反 | 1件（4.8倍超過） | **0件** | **解消** |
| 平均距離 | 55.5 km | **38.8 km** | -16.7 km |
| 最大距離 | 168.7 km | 104.8 km | -63.9 km |

**開設施設**: W01（さいたま）、W06（厚木）、W09（土浦）、W10（太田）

- 容量違反は解消されたが、コストは3.7倍に増加
- 4施設全てが稼働率95%以上（需要にピッタリの容量配分）
- 最大距離はまだ104.8km（カバレッジ要件の50kmを大幅超過）
- W06（厚木）とW10（太田）が選ばれたのは固定費が安いため（70万、55万）

### 改善2: P-median（施設数の感度分析）

| P | 総コスト(円/月) | 平均距離(km) | 最大距離(km) | 容量違反 |
|---|---------------|------------|------------|---------|
| 3 | 2,971,651 | **27.0** | 92.7 | 2件 |
| 4 | 4,060,810 | **24.6** | 92.7 | 1件 |
| 5 | 4,551,900 | **22.6** | **51.5** | 1件 |

- P=3: 距離は良いが容量違反2件（W01とW08で超過）
- P=4: W05（横浜金沢）追加で南部カバー改善。容量違反1件
- P=5: W09（土浦）追加で北東部カバー。最大距離が51.5kmまで改善
- P=5でも容量違反あり（P-medianは容量を考慮しない）

**施設数と距離のトレードオフ:**
- P=3→4: 距離-2.4km、コスト+109万円/月
- P=4→5: 距離-2.0km、コスト+49万円/月
- P=5が最もカバレッジが良い（51.5km）が、容量制約を満たさない

### 改善3: CFL + カバレッジ制約

50km超の割当を禁止 + 容量制約 + 最大5施設の制約を同時に課した。

| 指標 | 値 |
|------|-----|
| ステータス | **Infeasible**（実行不能） |
| 原因 | S21（水戸ショップ）が全候補から50km超 |

S21を除外すれば実行可能だが、要件として「全顧客カバー」が必要な場合は以下の対応が必要:
- 水戸近辺に候補施設を追加
- カバレッジ要件を52kmに緩和
- S21を別ルート（直送等）で対応

## 3. 推奨解: CFL ソルバー（4施設）

ベースラインと改善策の全体比較から、**CFL ソルバー（4施設開設）** を推奨する。

| 評価基準 | 全施設開設 | UFL | CFL | P-median(5) |
|---------|----------|-----|-----|-------------|
| 総コスト | 818万円 | 85万円 | **312万円** | 455万円 |
| 容量違反 | あり | あり | **なし** | あり |
| 平均距離 | 19.6km | 55.5km | 38.8km | 22.6km |
| 最大距離 | 51.5km | 168.7km | 104.8km | 51.5km |
| 実用性 | No | No | **Yes** | No |

**CFL が唯一の実用的な解（容量違反0件）。**

ただし最大距離104.8kmはカバレッジ要件を満たさないため、追加改善が必要:
1. 50km超の顧客をリストアップし、最寄り候補の追加を検討
2. または配送頻度を下げる（遠距離顧客は月2回→月1回等）

## 4. 改善効果のまとめ

```
ベースライン（UFL 1施設）
  容量: 800 units に 3,820 units → 4.8倍超過
  最大距離: 168.7 km → カバレッジ要件の3倍以上
  ↓
改善後（CFL 4施設）
  容量違反: 0件 ← 解消
  平均距離: 55.5km → 38.8km（-30%）
  コスト: 85万円 → 312万円（+267%、ただし容量を満たすための必要コスト）
```

## 5. 次のステップ

1. **追加候補地の検討**: 水戸・つくば周辺で50km要件を満たす候補地を追加
2. **感度分析**: 固定費±20%、需要±30%での解の安定性を検証
3. **多目的最適化**: コスト最小化 + 最大距離最小化のパレートフロントを算出
4. **実データでの検証**: Haversine距離をAPI距離に置き換えて精度向上
