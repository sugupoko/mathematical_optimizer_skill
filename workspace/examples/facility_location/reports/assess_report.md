# データ分析レポート（アセスメント）: 施設配置最適化

## 1. 問題概要

| 項目 | 値 |
|------|-----|
| 候補施設数 | 10箇所（関東広域） |
| 顧客数 | 30小売店（関東広域） |
| 総需要 | 3,820 units/月 |
| 総容量（全候補） | 10,200 units/月 |
| 需要/容量比 | 0.375（**容量に余裕あり**） |
| カバレッジ要件 | 全顧客が50km以内 |

## 2. 問題分類

- **問題タイプ**: 容量制約付き施設配置問題（CFL: Capacitated Facility Location）
- **主要な判断**: どの候補地に倉庫を開設し、各小売店をどの倉庫に割り当てるか
- **制約の難しさ**: 中程度（容量制約 + カバレッジ制約の組合せ）
- **問題規模**: 小規模（10施設×30顧客、変数310個）→ ソルバーで厳密解が得られる

### 3.2 顧客の地理的分布

需要の地理的集中を分析すると以下の3クラスタが見える:

1. **都心部クラスタ**（S01-S06, S22-S25）: 新宿・渋谷・池袋・銀座周辺、需要合計1,370 units
2. **西部クラスタ**（S07-S12, S26-S28）: 立川・町田・横浜・川崎周辺、需要合計960 units
3. **北部・東部クラスタ**（S13-S21, S29-S30）: 大宮・千葉・柏・つくば周辺、需要合計1,490 units

### 3.3 距離の特徴

- 全顧客の最寄り候補施設までの平均距離: 19.7 km
- 最寄り候補施設までの最大距離: 51.5 km（S21 水戸ショップ → W09 土浦倉庫）
- 50km以内のカバレッジ: 29/30（97%）

**要注意顧客: S21 水戸ショップ**
- 全10候補から50km超
- 最寄り: W09 土浦倉庫（51.5km）
- 50kmカバレッジ要件を満たすには、水戸近辺に追加候補が必要か、要件を51.5kmに緩和する必要がある

### 3.4 コスト構造

| 分析項目 | 値 |
|---------|-----|
| 全施設開設時の固定費 | 810万円/月 |
| 最安施設1つの固定費 | 50万円/月（土浦倉庫） |
| 輸送コスト単価 | 150円/km |
| 全顧客を1施設で賄う場合の輸送費 | 約25万円/月 |

固定費が輸送費に対して圧倒的に大きい。
→ ソルバーは施設数を最小化する方向に強いバイアスがかかる。
→ 容量制約なしでは1施設に全集約される可能性が高い。

## 4. 仮説

1. **容量制約なし（UFL）では1-2施設に集約される**: 固定費が支配的なため
2. **容量制約付き（CFL）では3-4施設が必要**: 総需要3,820 unitsを満たすには最低3施設
3. **50kmカバレッジは29/30が上限**: S21をカバーするには追加候補地 or 要件緩和が必要
4. **都心部に近い施設が重要**: 需要の36%が都心部に集中

## 5. 推奨アプローチ

1. **まずUFLでベースライン**: 固定費+輸送費の最小化（容量無視）
2. **次にCFL**: 容量制約を追加して現実的な解を求める
3. **P-medianで感度分析**: P=3,4,5でコスト・距離のトレードオフを可視化
4. **カバレッジ制約付きCFL**: 50km上限を組み込んだ最終解

## 6. 不足情報

| 情報 | 重要度 | 影響 |
|------|--------|------|
| 実際の道路距離（API） | 中 | Haversine×1.3は概算。特に山間部で誤差大 |
| 配送頻度 | 高 | 月1回 vs 週1回で輸送コストが大幅に変わる |
| 需要の季節変動 | 中 | ピーク時に容量不足にならないか |
| 候補地の増減 | 高 | 水戸近辺の追加候補があればカバレッジ改善 |
| 初期投資コスト | 中 | 固定費に含まれない場合は投資回収年数の検討が必要 |
